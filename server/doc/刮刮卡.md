
#### 初始化

`url`: `http://192.168.3.5:8080/game/scratch-card`

`params`:

```json
{
  "id": "zhangsan",
  "isScratch":true
}
```
> 请求参数  
> `id` 微信id  
> `isScratch` 是否为刮卡请求 (1, -1)  

`return`:
```json
{
    "curUserGroup": 1,
    "lastList": [
        {
            "money": 9,
            "nickname": "test_user_9",
            "id": 9
        }
    ],
    "partnerStatus": false,
    "todayList": [
        {
            "money": 0,
            "nickname": "test_user_0",
            "id": 0
        },
        {
            "money": 1,
            "nickname": "test_user_1",
            "id": 1
        },
        {
            "money": 7,
            "nickname": "test_user_7",
            "id": 7
        },
        {
            "money": 8,
            "nickname": "test_user_8",
            "id": 8
        }
    ],
    "topList": [
        {

```

> 返回参数:  
> `partnerStatus` 用户当日刮卡状态 -1表示未刮卡，0表示正在刮卡，1表示已经刮卡  
> `curUserGroup` 当前用户组  
> `lastList` 今日最低   
> `topList` 今日最高  
> `todayList` 今日所有刮卡信息  
> `totalNum` 今日收到总捐款  
> `totalTop` 本月最佳
